Module depile::ir::instr::basic[−][src]

Expand description

Basic instructions for “raw” three-address code, with 1-1 correspondence to valid input texts.

Also repurposed for block-level instructions. These are just like basic instructions, but with the exceptions that all destinations in Branching instructions are interpreted as block indices instead of instruction indices, and that the marker EntryPc shall never be used.
