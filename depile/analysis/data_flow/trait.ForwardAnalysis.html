fn transfer_function(
    block_idx: usize,
    block: &Block<K>,
    input: &Self,
    output: &mut Self
) -> bool

Transfer function f such that OUT[B] = f(IN[B]).

IN[B] is provided as input, OUT[B] provided for update, the returned bool indicates whether or not the OUT[B] changed. For monotone frameworks, IN[B] = f(OUT[B]) can be substituted with OUT[B] = OUT[B] ⊓ f(IN[B]). This allows using JoinSemiLattice::join_assign on OUT[B].
