fn transfer_function(
    block_idx: usize,
    block: &Block<K>,
    input: &mut Self,
    output: &Self
) -> bool

Transfer function f such that IN[B] = f(OUT[B]).

OUT[B] is provided as input, IN[B] provided for update, the returned bool indicates whether or not the IN[B] changed. For monotone frameworks, IN[B] = f(OUT[B]) can be substituted with IN[B] = IN[B] ⊓ f(OUT[B]). This allows using JoinSemiLattice::join_assign on IN[B].
